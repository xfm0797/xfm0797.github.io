图床配置

从 2.0.0 开始, Qexo 支持了模块化的图床接入方法, 先前的 S3、FTP、Custom 文档合并至此

远程 API

Qexo 提供了自定义 API 图床功能, 在配置完成图床设置后即可在文章/页面编辑界面上传图片

API 地址

图床图片上传的 API

JSON 路径

自定义请求头

POST 请求时附带的请求头, 需要标准 JSON 格式, 若不需要请留空

自定义 BODY

POST 请求时额外的请求主体, 需要标准 JSON 格式, 若不需要请留空

自定义前缀

返回 URL 所需要添加的前缀, 若不需要请留空

删除 API
